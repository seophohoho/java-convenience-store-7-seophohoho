# 기능 목록

## 초기화

- `products.md`, `promotions.md` 파일 읽고, 물품 저장

## 주문 받기

- 구매할 상품이 올바른지 검사
- 상품 수량이 올바른지 검사

## 주문 저장

- 주문 데이터 저장
- 각 주문들의 데이터가 올바른지 파악
    - 존재하지 않은 상품을 입력한 경우,
    - 구매 수량이 재고 수량을 초과한 경우,
    - 기타 잘못된 경우,

## 프로모션 처리

- 금일 날짜가 프로모션 기간인지 체크
- 프로모션 로직 처리
